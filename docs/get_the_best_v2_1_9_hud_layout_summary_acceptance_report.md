# 《壮志凌云 / Get The Best》V2-1.9 HUD 布局与月报交互验收报告

日期：2026-05-23

## 本轮目标

- 修复底部经营工具栏打开菜单时整块面板被撑大的布局问题。
- 将日期从底部右侧移动到顶部居中，并从“第 1 月 第 1 天”调整为“第1年 第1月 第1天”。
- 在右上角增加轻量核心概要，显示阶段与用户评分占位信息，后续由 C# Core 结果驱动。
- 压缩底部经营工具栏宽度，避免“用户”和“出售”后方留出过大空白，同时保证时间倍率条完整显示。
- 月报窗口改为更小的居中弹窗，只能点击“继续经营”关闭；打开月报时禁用鼠标贴边移动。

## 实现范围

- `BuildModeHudController`：
  - 底部工具栏保持固定紧凑高度。
  - 建造、设施、员工、管理部、发行部、统计菜单改为按钮上方浮层，不再撑开底栏。
- `BusinessCalendarHudController`：
  - 增加年份字段与 12 月进年逻辑。
  - 默认日期显示为“第1年 第1月 第1天”。
- `CoreSummaryHudController`：
  - 新增右上角概要面板。
  - 当前展示阶段与用户评分占位，数据来自 `CoreOfficeSimulationResult`。
- `MonthlyReportHudController`：
  - 尺寸调整为 `460x430`。
  - 鼠标左键只吞掉月报区域输入，不再直接关闭月报。
- `OfficeCamera3DController`：
  - 月报显示时停止贴边移动，避免阅读月报时镜头漂移。

## 验收结果

- 已新增 `test_get_the_best_v2_1_9_hud_layout_report_and_summary_fix`。
- 已更新旧阶段中与本轮需求冲突的布局断言。
- 已通过 Godot C# Debug 编译。
- 已通过 Godot MCP 实机验证：
  - 底部工具栏保持 `542x50`，打开建造菜单时只显示 `BuildModeMenuPopup` 浮层。
  - 顶部居中日历可显示年份、月份与日期，月报等待态显示年份与月份。
  - 右上角核心概要显示阶段与用户评分。
  - 月报显示为居中小窗，非按钮点击不会关闭；点击“继续经营”后关闭。
  - `get_errors` 返回 0 error。
